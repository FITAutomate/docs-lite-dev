# Content Guide

## Authoring baseline

- One purpose per page.
- Use plain language and short sections.
- Keep headings functional and searchable.
- Prefer procedure flow over narrative.

## Frontmatter baseline

Use this minimum:

```md
---
title: Example Page
description: One-line summary.
sidebar:
  label: Example
---
```

## Naming baseline

- Use lowercase file names with hyphens.
- Reserve `index.md` for section landing pages.
