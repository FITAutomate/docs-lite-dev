# Docs Support Pack

Support documentation for FIT Docs Lite template users.

This folder is not the rendered docs site content. The site content lives in:

`web/src/content/docs/`

Use this folder for onboarding guides, publishing SOPs, and maintenance notes.

## Policies

- **Branching model:** `docs/branching-model.md` - `main` vs `docs-lite-showcase` vs `dev/*`, merge rules, and showcase hosting.
- **Demo hosting:** `docs/demo-hosting-policy.md` - GitHub Pages default for demos/showcases and exception rules.
