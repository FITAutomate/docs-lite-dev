Search is the fastest way to navigate when you already know what task you need to complete.

## When to use search

- you know part of a page title but not its exact location
- you need a specific procedure quickly
- you want to confirm whether a topic already exists before drafting a new page

## Simple search workflow

1. start with the main noun for your task
2. open the best match
3. if needed, adjust terms and try again
4. copy the page link when sharing with teammates

Next step: use [Find - How](/help/search/find-how/) for practical query patterns.
