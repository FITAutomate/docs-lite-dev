FIT Docs Lite is a template for teams that need useful documentation quickly without building a full docs system from scratch.

Use this library to:

- orient new contributors
- write and improve pages in small steps
- track changes in GitHub
- deploy updates through your team's configured publish workflow

If you are new here, begin with [Getting Started](/getting-started/). If you want the big picture first, read [FIT Intro](/fit-intro/).
