Use this section when you are creating new pages or improving existing ones.

## Writing model in FIT Docs Lite

- write in small, readable sections
- prefer clear headings over long paragraphs
- improve existing pages before creating duplicates
- make changes that are easy to review and publish

## Choose your path

- [Drafting](/writing/drafting/) for new content or larger updates
- [Live Page Edits](/writing/live-page-edits/) for small safe fixes on existing pages

After writing, continue to [Publishing](/publishing/) to review and release your changes.
