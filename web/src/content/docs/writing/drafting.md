Drafting is where you turn rough notes into clear, useful pages.

## Draft in small blocks

Use this structure:

1. one clear page purpose
2. short section headings
3. step-by-step instructions
4. links to related pages

## Beginner-safe drafting workflow

1. Search for an existing page first.
2. If one exists, improve it instead of duplicating.
3. If not, create a new page with a clear title.
4. Keep early drafts simple, then refine wording.
5. Check formatting and links before committing.

## Writing style for this template

- prefer plain language over technical terms
- explain actions in the order users take them
- keep examples generic so the template stays reusable

For small fixes on published pages, use [Live Page Edits](/writing/live-page-edits/).
